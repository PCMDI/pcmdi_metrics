CLIVAR 2020 ENSO Metrics¶

Overview¶

To be used within the PMP, the CLIVAR ENSO metrics first need to be installed in the user’s local PMP environement. The most practical way to do this is via the CLIVAR GitHub repository (https://github.com/CLIVAR-PRP/ENSO_metrics).
